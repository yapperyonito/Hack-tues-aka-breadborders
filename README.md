This project is a smart trash can that rewards users for throwing away trash. Every time trash is placed inside, the system increases a score. When the score reaches a certain number, a connected candy machine dispenses candy as a reward.
A sensor detects when trash is thrown into the bin.
Each detected action increases a score by 1.
The system continuously checks if the score has reached a set threshold.
Once the threshold is reached, a candy dispenser is activated.
The score is then reset (or continues, depending on setup).
The goal of this project is to encourage people to keep their environment clean by rewarding positive behavior in a fun and engaging way.
